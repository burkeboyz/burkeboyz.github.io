Alerts

Alerts are created with the .alert class, followed by a contextual color classes:

Success! light green This alert box is light green
Info! light blue This alert box is light blue.
Warning! yellow This alert box is yellow.
Danger! faint red This alert box is a faint red.
Primary! faint blue This alert box is a faint blue.
Secondary! faint gray This alert box is a faint gray.
Dark! This alert box is Dark grey .
Light! This alert box has no color.
